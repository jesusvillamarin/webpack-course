Creando un servidor de desarrollo con webpack-dev-server

Para que se este refrescando cada modificación realizada al entry point ./src/js/index.js.
Es necesario declarar la etiqueta script apuntando al bundle que genera la ruta del servidor de desarrollo, que en este caso seria http://localhost:8080/bundle.js


Si en el archivo script el puerto que se maneja es el 2904 es debido a que el webpack.config.js modificamos la configuración del puerto, webpack-dev-server por defecto asigna el puerto 8080
